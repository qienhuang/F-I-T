## Universal Essays

This folder contains longform FIT essays aimed at broad, cross-domain questions.

These texts are interpretive and explanatory. They are not FIT core specifications.

### Included

- [Evolution as Boundary Dynamics](evolution_as_boundary_dynamics.md)  
  A bounded longform essay arguing that major evolutionary transitions can often be understood as changes in what a system is able to internalize and preserve as structure.

### Reading boundary

These essays should be read as:

- longform interpretation
- concept-building for broad readers
- core-adjacent, not core-defining

In particular, [Evolution as Boundary Dynamics](evolution_as_boundary_dynamics.md) should not be read as:

- a universal law
- a completed bridge theorem
- proof that one experimentally isolated mechanism already unifies learning, biology, science, and institutions
