记号：相位写作 `Φ₁/Φ₂/Φ₃`（在文件名/代码中用 `Phi1/Phi2/Phi3`）。

## 目的

MCC-6 断言：晚期（Φ₃）系统呈现不可逆性。但不可逆性不是二值；它有程度。本文定义一个**稳定性判据族**，使 Φ₃ 的判定在 EST 下可操作、可审计。

## 核心问题

> 什么时候系统算“进入 Φ₃”？“稳定到什么程度”才算够？

在 EST 下，这本质是测量问题：依赖于你声明的 estimators、观测窗与容差。

## 稳定性判据族（SCF）

我们定义三档判据，按严格程度递增。

### SC-1：结构持久性（最弱）

**定义**：在声明的估计器下，主要约束结构在 `tau_persist` 时长内持续存在，且无显著回退。

**最小操作化检查**：

在观测窗 `[t0, t0 + tau_persist]` 内，声明一个约束代理 `C(t)` 与容差 `epsilon`。

SC-1 成立当且仅当：
- 对窗内所有 `t`，`abs(C(t) - C(t0)) <= epsilon`

**用途**：快速稳定性检查；是进入 Φ₃ 的必要条件，但不足以证明“深稳定”。

### SC-2：扰动韧性（中等）

**定义**：在一个有界扰动后，系统能在 `tau_recover` 内回到原约束结构邻域。

**最小操作化检查**：

声明：
- 扰动预算 `delta`（多大的冲击算“有界”）
- 恢复时间预算 `tau_recover`
- 同一个约束代理 `C(t)` 与容差 `epsilon`

SC-2 成立当且仅当：
- 在施加幅度 `<= delta` 的扰动后，存在 `tr <= tau_recover` 使得 `abs(C(t0 + tr) - C(t0)) <= epsilon`

**用途**：评估韧性/吸引子强度；适用于有持续环境扰动的系统。

### SC-3：可转移稳定（最强）

**定义**：约束结构可在“新载体/新上下文”中保持，并仍满足 SC-1。

**最小操作化检查**：

SC-3 成立当且仅当：
- 在载体/上下文 `S1` 上满足 SC-1，且
- 在声明的迁移协议下转移到 `S2` 后，`S2` 上仍满足 SC-1

**用途**：区分真正的“协调稳定”（Φ₃）与“载体锁定”的晚期 Φ₂。

## 判据速查表

| 判据 | 测什么 | 常见失败模式 |
|------|--------|--------------|
| SC-1 | 基本持久性 | 仍处于 Φ₂ 或更早 |
| SC-2 | 吸引子强度 | 处于脆弱 Φ₃（浅稳定） |
| SC-3 | 可转移性 | 尚未达到深稳定（nirvana-like） |

## Φ₃ 子区间（分类）

| 子区间 | 满足判据 | 说明 |
|--------|----------|------|
| Φ₃-（边缘） | 仅 SC-1 | 刚进入 Φ₃，易退回 |
| Φ₃（标准） | SC-1 + SC-2 | 稳定吸引子，具备有界韧性 |
| Φ₃+（深） | SC-1 + SC-2 + SC-3 | 可转移稳定，nirvana-like |

## 操作化例子（示意）

### 例 1：Grokking（学习系统）

- SC-1：测试集准确率 >= 95% 持续 `tau_persist` 个 epoch
- SC-2：有界权重扰动后，在 `tau_recover` 内恢复
- SC-3：特征可迁移到相关任务，并仍满足 SC-1

## 与 MCC-6 的关系

MCC-6（晚期不可逆性）可按以下方式落地评估：

- 在声明的估计器族下，若满足 **SC-1 + SC-2**，可标注为 **Φ₃（标准）**。
- 只有进一步满足 **SC-3**，才可标注为 **Φ₃+（深）**。
